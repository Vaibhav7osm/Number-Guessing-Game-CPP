# Number-Guessing-Game-CPP
A simple console-based game written in C++ where the computer randomly selects a number and the player must guess it.

## Features
- Three difficulty levels (Easy, Medium, Hard)
- Random number generation
- High / Low hints for guesses
- Score system
- Maximum attempt limit
- Option to play again

## Concepts Used
- Functions
- Loops
- Conditional statements
- Random numbers
- User input/output
